# 📋 TASK-002-003: UI de Productos — Campos de IVA

**TASK-002-003**: Añadir selector de IVA al formulario de productos
**Épica**: EPIC-002

**Duración estimada**: 1.5h
**Prioridad**: 🔴 CRÍTICO
**Estado**: ⏳ Pendiente

## 📖 Descripción
El formulario de crear/editar producto necesita dos controles nuevos:
1. **Selector de tasa de IVA**: 0% (Exento), 8%, 16%, Personalizado.
2. **Toggle**: "El precio ya incluye IVA" / "IVA se agrega al total".

Ambos valores se persisten en SQLite via los handlers existentes `products:create` y `products:update`.

---

## 🎯 Objetivo
El formulario de producto en `/products` tiene los campos de IVA funcionales,
con valores por defecto leídos de `settings.tax_rate_default`.

---

## 📝 Pasos de Implementación

### Paso 1: Leer el IVA default de Settings al abrir el formulario
**Archivo:** Componente de formulario de producto en `src/app/products/`
Al abrir el modal de "Nuevo Producto", pre-cargar `taxRate` con el valor de `settings.tax_rate_default`.

### Paso 2: Añadir selector de tasa
Usar `Select` de Shadcn con opciones: Exento (0), 8% (800), 16% (1600), Personalizado.
Si es "Personalizado", mostrar un `Input` numérico.

### Paso 3: Añadir toggle "IVA incluido"
Usar `Switch` de Shadcn para `taxIncluded`.

### Paso 4: Mostrar preview del precio
Debajo del campo de precio, mostrar en tiempo real el desglose calculado con `calculateItemTax`.

---

## 💻 Código de Referencia

### Fragmento para el formulario (añadir a la sección de precio):

```tsx
import { calculateItemTax, formatCents, pctToRate } from "@/lib/services/tax";
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from "@/components/ui/select";
import { Switch } from "@/components/ui/switch";
import { Label } from "@/components/ui/label";

// En el estado del formulario:
const [taxRate, setTaxRate] = useState(1600); // default desde settings
const [taxIncluded, setTaxIncluded] = useState(true);
const [taxPreset, setTaxPreset] = useState<"0" | "800" | "1600" | "custom">("1600");

// Preview del IVA (calculado en tiempo real):
const priceInCents = Math.round(parseFloat(priceInput || "0") * 100);
const taxPreview = calculateItemTax(priceInCents, taxRate, taxIncluded);

// JSX a añadir en el formulario:
<div className="space-y-4 pt-4 border-t border-border/50">
  <p className="text-[10px] font-black uppercase tracking-widest text-muted-foreground">
    Configuración de Impuesto
  </p>

  {/* Selector de tasa */}
  <div className="space-y-2">
    <Label htmlFor="taxRate" className="text-[10px] font-black uppercase tracking-widest text-muted-foreground">
      Tasa de IVA
    </Label>
    <Select
      value={taxPreset}
      onValueChange={(val) => {
        setTaxPreset(val as typeof taxPreset);
        if (val !== "custom") setTaxRate(parseInt(val));
      }}
    >
      <SelectTrigger id="taxRate" className="h-11">
        <SelectValue />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="0">Exento (0%)</SelectItem>
        <SelectItem value="800">8% (Frontera)</SelectItem>
        <SelectItem value="1600">16% (Estándar)</SelectItem>
        <SelectItem value="custom">Personalizado</SelectItem>
      </SelectContent>
    </Select>

    {taxPreset === "custom" && (
      <div className="flex items-center gap-2">
        <Input
          type="number"
          min="0" max="100" step="0.01"
          placeholder="Ej: 10.5"
          className="h-11 w-32"
          onChange={(e) => setTaxRate(pctToRate(parseFloat(e.target.value || "0")))}
        />
        <span className="text-sm text-muted-foreground">%</span>
      </div>
    )}
  </div>

  {/* Toggle IVA incluido */}
  <div className="flex items-center justify-between p-4 border rounded-xl bg-muted/20">
    <div>
      <p className="text-sm font-bold">El precio ya incluye IVA</p>
      <p className="text-xs text-muted-foreground mt-0.5">
        {taxIncluded
          ? "El IVA se extrae del precio que capturaste"
          : "El IVA se sumará al precio al momento de vender"}
      </p>
    </div>
    <Switch
      checked={taxIncluded}
      onCheckedChange={setTaxIncluded}
    />
  </div>

  {/* Preview del desglose */}
  {priceInCents > 0 && taxRate > 0 && (
    <div className="p-3 rounded-lg bg-primary/5 border border-primary/10 text-xs space-y-1">
      <p className="font-bold text-[10px] uppercase tracking-widest text-muted-foreground">
        Vista previa del desglose
      </p>
      <div className="flex justify-between">
        <span>Precio base:</span>
        <span className="font-mono font-bold">{formatCents(taxPreview.basePrice)}</span>
      </div>
      <div className="flex justify-between text-amber-600">
        <span>IVA ({taxRate / 100}%):</span>
        <span className="font-mono font-bold">+{formatCents(taxPreview.taxAmount)}</span>
      </div>
      <div className="flex justify-between border-t pt-1 font-black">
        <span>Total a cobrar:</span>
        <span className="font-mono">{formatCents(taxPreview.total)}</span>
      </div>
    </div>
  )}
</div>
```
